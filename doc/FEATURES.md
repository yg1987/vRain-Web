## 1. 命令行接口

### 1.1 基本命令

```bash
perl vrain.pl -b <book_id> [-f <from>] [-t <to>] [-z <pages>] [-c] [-v] [-h]
```

### 1.2 参数详解

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `-b <ID>` | 必填 | — | 书籍编号，对应 `books/<ID>/` 目录 |
| `-f <N>` | 整数 | `1` | 起始文本文件索引（按文件名排序后的序号，非文件名数字） |
| `-t <N>` | 整数 | `1` | 结束文本文件索引 |
| `-z <N>` | 整数 | — | 测试模式：仅生成 N 页，文件名追加 `_test` 后缀 |
| `-c` | 开关 | 关闭 | 启用 Ghostscript 压缩 PDF |
| `-v` | 开关 | 关闭 | 详细模式：逐字输出 `[页号/字符索引] 字符 -> 字体文件` |
| `-h` | 开关 | — | 显示帮助信息并退出 |

### 1.3 输入验证

| 检查项 | 行为 |
|--------|------|
| `books/<ID>/` 目录存在 | 不存在则报错退出 |
| `books/<ID>/text/` 目录存在 | 不存在则报错退出 |
| `books/<ID>/book.cfg` 文件存在 | 不存在则报错退出 |
| `canvas/<canvas_id>.cfg` 文件存在 | 不存在则报错退出 |
| `canvas/<canvas_id>.jpg` 文件存在 | 不存在则报错退出 |
| `font1` 已定义 | 未定义则报错退出 |
| `font1`~`font5` 文件存在于 `fonts/` | 缺失则报错退出 |
| 多栏模式: `row_num % multirows_num == 0` | 不整除则报错退出 |

### 1.4 输出命名

| 模式 | 文件名格式 |
|------|-----------|
| 正常 | `《<title>》文本<from>至<to>.pdf` |
| 测试 | `《<title>》文本<from>至<to>_test.pdf` |
| 压缩后 | `<原文件名>_已压缩.pdf` |

## 2. 排版与分页

### 2.1 基本排版方向

| 特性 | 说明 |
|------|------|
| 阅读方向 | 从上到下（竖排），从右到左（列序） |
| 字符定位 | 逐字计算精确坐标，非行内流式排版 |
| 版心居中 | 版心左右对称，支持中缝（叶心）分隔 |

### 2.2 网格布局

| 参数 | 来源 | 说明 |
|------|------|------|
| `col_width` | 计算值 | `(canvas_width - ml - mr - leaf_center_width) / col_num` |
| `row_height` | 计算值 | `(canvas_height - mt - mb) / row_num` |
| 列分配逻辑 | 内部算法 | 左半列使用左边距定位，右半列额外偏移 `leaf_center_width` |
| `row_delta_y` | book.cfg | 调整每列最后一个字符的纵向位置，默认 `8` |
| `if_tpcenter` | canvas.cfg | 标题居中开关：`1`=居中，`0`=左对齐 |

### 2.3 分页控制

| 方式 | 触发机制 | 说明 |
|------|----------|------|
| 自动翻页 | 字符计数达到 `page_chars_num` | 每页容量 = `col_num × row_num` |
| 强制换页 `%` | 文本中标记符 | 跳过剩余字符填满列高，开启新页；已在页首时重置为 `0` |
| 半页跳 `$` | 文本中标记符 | 若 `pcnt < 半页容量` → 跳到半页位置；否则跳到满页位置 |
| 末列跳 `&` | 文本中标记符 | 跳到当前页最后一列 |
| 多栏跳 `^` | 文本中标记符 (vrain.pl 独有) | 在多栏模式下跳到下一个栏块 |
| 段落缩进 `T` | 文本开头标记 (vrain.pl 独有) | 下一字符下沉一行，边框通过填充矩形视觉延伸 |

### 2.4 页容量计算

| 模式 | 计算公式 |
|------|----------|
| 标准模式 | `col_num × row_num` |
| 多栏模式 | `col_num × row_num × multirows_num` |

## 3. 字体与排版

### 3.1 多级字体回退系统

| 配置项 | 说明 |
|--------|------|
| `font1` ~ `font5` | 最多 5 级字体文件名，存放在 `fonts/` 目录 |
| `text_fonts_array` | 正文字体的优先顺序，如 `"123"` 表示先试 font1，再 font2，再 font3 |
| `comment_fonts_array` | 批注字体的优先顺序 |
| 检测方式 | 使用 `Font::FreeType::glyph_from_char()` 逐字检测字形是否存在 |
| 回退结果 | 无匹配字体时渲染为 `□` (方框占位符) |

### 3.2 字体属性配置

| 配置项 | 说明 |
|--------|------|
| `text_font<N>_size` | 正文字体大小（每级独立） |
| `comment_font<N>_size` | 批注字体大小（每级独立） |
| `text_font<N>_color` | 正文颜色（每级独立），默认 `black` |
| `comment_font_color` | 批注颜色，默认 `black` |
| `font<N>_rotate` | 每级字体的旋转角度 |
| 拉丁字母自动旋转 | `[a-z]` 及带音调字母自动旋转 `-90°`，位置偏移 `fx+=fsize/4, fy+=fsize/2` |

### 3.3 字体度量校准 (`if_font_metric_adjust`)

启用方式: `if_font_metric_adjust=1`（book.cfg）

| 阶段 | 机制 |
|------|------|
| Phase 1 (逐字级) | 以参考字符 `"国"` 为基准，用 FreeType 测量各字体实际字形高度，计算缩放比例 |
| Phase 2 (面部级回退) | 对不包含参考字符的字体（如 CJK Extension 平面），使用 `ascender - descender` 面级指标 + 主字体校准因子 |
| 缓存机制 | `%face_cache` 缓存 FreeType face 对象，`%font_scale` 缓存缩放比例 |
| 自动应用 | 所有文字渲染时自动乘以 `$font_scale{$fontname}` |

### 3.4 回退加粗模拟 (`if_fallback_bold`)

启用方式: `if_fallback_bold=1`（book.cfg）

| 配置项 | 说明 |
|--------|------|
| `if_fallback_bold` | 启用回退字体描边模拟加粗 |
| `fallback_bold_stroke_width` | 描边宽度 |
| 实现方式 | PDF `-render => 2, -strokecolor => $fcolor` 叠加描边 |

### 3.5 中文数字转换

| 功能 | 范围 | 说明 |
|------|------|------|
| `get_pid_zh()` | 页码数字 | `0→〇`, `10→十`, `11~19→十一~十九`, `20~90→二十~九十` |
| `db/num2zh_jid.txt` | 章节编号 | 阿拉伯数字→繁体中文数字映射表（0~400+） |

### 3.6 字体文件支持

| 格式 | 支持 |
|------|------|
| `.ttf` (TrueType) | ✅ |
| `.otf` (OpenType) | ✅ |
| 内置字体 | `qiji-combo.ttf` (~41MB), `HanaMinA/B.ttf`, 约 20 种中文字体 |

## 4. 标点符号处理

### 4.1 标点替换 (`exp_replace_comma`)

| 配置 | 说明 |
|------|------|
| 格式 | 管道分隔的成对字符: `from1|to1|from2|to2` |
| 示例 | `,，|.。|:：` 将半角逗号/句号/冒号替换为全角 |
| 转义 | `. ! ? ( ) [ ] -` 等特殊字符自动转义 |

### 4.2 数字替换 (`exp_replace_number`)

| 配置 | 说明 |
|------|------|
| 功能 | 阿拉伯数字替换为中文数字 |
| 覆盖范围 | `0~9` 半角 + 全角 `０~９` |
| 示例 | `1一|2二|...|0〇|１一|２二|...` |

### 4.3 标点删除 (`exp_delete_comma`)

| 配置 | 说明 |
|------|------|
| 功能 | 完全删除指定的标点字符 |
| 格式 | 管道分隔列表 |
| 常见删除 | `．`, ` `, `-`, `─`, `〖`, `〗`, `(`, `)`, `‧`, `·`, `•`, `「`, `」`, `『`, `』`, `－`, `\|` |

### 4.4 无标点模式 (`if_nocomma`)

| 配置 | 说明 |
|------|------|
| 启用 | `if_nocomma=1` |
| 正则 | `exp_nocomma` 匹配的字符全部删除 |
| 用途 | 文言文等不需要现代标点的文本 |

### 4.5 统一句号模式 (`if_onlyperiod`)

| 配置 | 说明 |
|------|------|
| 启用 | `if_onlyperiod=1` |
| 行为 | 将 `exp_onlyperiod` 匹配的所有标点统一转为 `。`（句号） |
| 规则 | 连续多个句号合并为一个，删除开头句号 |

### 4.6 不占位标点 (`text_comma_nop`)

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `text_comma_nop` | — | 不占字符位置的标点正则 |
| `text_comma_nop_size` | `1.1` | 缩放比例 |
| `text_comma_nop_x` | `0.45` | 水平偏移系数 |
| `text_comma_nop_y` | `0.5` | 纵向偏移系数 |
| 边界安全 | 底边 clamped | `fy` 不低于 `margins_bottom + 5~10` |
| 省略号/破折号 | 特殊处理 | `fy += fsize/2` 拉长 |
| 页尾处理 | 弹出到新页 | 在页底时，不占位标点所在字符被推入下一页 |

### 4.7 旋转标点 (`text_comma_90`)

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `text_comma_90` | — | 需要旋转 `-90°` 的标点 |
| `text_comma_90_size` | `0.8` | 缩放比例 |
| `text_comma_90_x` | `0.35` | 水平偏移 |
| `text_comma_90_y` | `0.65` | 纵向偏移 |

### 4.8 批注标点独立配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `comment_comma_nop` | — | 批注不占位标点（半字符位置） |
| `comment_comma_nop_size` | `0.6` | 缩放比例 |
| `comment_comma_nop_x` | `0.6` | 水平偏移 |
| `comment_comma_nop_y` | `0.12` | 纵向偏移 |
| `comment_comma_90` | — | 批注旋转标点 |

## 5. 标记语法与特殊字符

### 5.1 正文标记

| 标记符 | 语法 | 启用条件 | 效果 |
|--------|------|----------|------|
| 批注 | `【text】` | 始终 | 夹批双列小字，半字符宽，支持跨列/跨页延续 |
| 强制换页 | `%` | 始终 | 强制翻页 |
| 半页跳 | `$` | 始终 | 向前跳半页或满页 |
| 末列跳 | `&` | 始终 | 跳到当前页最后一列 |
| 空格 | `@` | 始终 | 插入空格 |
| 段落缩进 | `T` | 始终 (vrain.pl) | 下一字符下沉一行 |
| 多栏跳 | `^` | 始终 (vrain.pl) | 多栏模式下跳到下一栏块 |

### 5.2 装饰标记

| 标记符 | 语法 | 启用条件 | 效果 |
|--------|------|----------|------|
| 书名号线 | `《text》` | `if_tag_bookline=1` | 文字左侧绘制正弦波浪装饰线 |
| 圆角框 | `〔text〕` | `if_tag_rectframe=1` | 圆角矩形包围文字（单层/双层边框） |
| 圆圈 | `〈text〉` | `if_tag_circleframe=1` | 圆形包围文字 |
| 字体放大 | `（text）` | `if_tag_textzoom=1` | 按 `text_zoom` 比例放大文字 |
| 圈注 | `｛text｝` | `if_tag_circlenote=1` | 字符右侧绘制强调小圆 |
| 顿点注 | `＜text＞` | `if_tag_pointnote=1` | 字符右侧绘制顿号 `、` |
| 行注 | `［text］` | `if_tag_linenote=1` | 字符右侧绘制竖线 |

### 5.3 标记参数配置

| 标记 | 可调参数 |
|------|----------|
| 书名号线 | `book_line_width` (线宽), `book_line_color` (颜色) |
| 圆角框 | `rect_type` (0=双层, 1=单层), `rect_bcolor` (边框色), `rect_fcolor` (填充色), 各尺寸偏移参数 |
| 圆圈 | `circle_type` (0=双层, 1=单层), `circle_bcolor`, `circle_fcolor`, `text_circle_y/r/f` |
| 字体放大 | `text_zoom` (默认 1.1) |
| 圈注 | `text_note_ox/y/r/w/c` (偏移/半径/线宽/颜色) |
| 顿点注 | `text_note_px/py/ps/pc` (偏移/大小/颜色) |
| 行注 | `text_note_lx/ly/lw/lc` (偏移/线宽/颜色) |

### 5.4 页尾标记处理

| 页尾遇到 | 行为 |
|----------|------|
| 标记开始符 (`《`, `〔`, `（`, `〈`, `｛`, `＜`, `［`) | 推回字符数组，在新页继续处理 |
| 标记结束符 (`》`, `〕`, `）`, `〉`, `｝`, `＞`, `］`) | 重置对应标记标志 |
| 不占位标点 | 在当前页末尾渲染 |
| 普通字符 | 推回，在新页开始 |

### 5.5 批注中的嵌套标记

夹批 `【】` 内支持的标记: `《》` (书名号), `〔〕` (圆角框), `〈〉` (圆圈)  
不支持: `｛｝`, `＜＞`, `［］`

## 6. 页面元素

### 6.1 版心标题

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `title_font_size` | — | 标题字号 |
| `title_font_color` | — | 标题颜色 |
| `title_y` | — | 标题纵向位置 |

| `pager_font_size` | — | 页码字号 |

| `pager_y` | — | 页码纵向位置 |

| 特性 | 说明 |
|------|------|

### 6.3 封面页

| 模式 | 条件 | 行为 |
|------|------|------|
| 自定义封面 | `books/<ID>/cover.jpg` 存在 | 直接嵌入为第一页 |

|------|------|

| 装饰横线 | 中线两侧 `±50px` 范围内，每隔 `200px` 一条 |
| 粗中线 | `#f2f2f2`，线宽 `20` |
| 书名 | 竖排，`cover_title_font_size` (默认 120)，`cover_title_y` (默认 200) |
| 作者 | 竖排，`cover_author_font_size` (默认 60)，`cover_author_y` (默认 600) |
| 工作室名 | 横排，位置 `($plx-300, 60)`，字号 `30` |
| 字体颜色 | `cover_font_color` (默认 `black`) |

## 7. 背景画布渲染

### 7.1 画布配置解析

| 规则 | 说明 |
|------|------|
| 空白行 | 跳过 |
| `#` 开头的行 | 注释，跳过 |
| 内联注释 | 不含 `=#` 的行，`#` 之后内容被删除 |
| 空白字符 | 全部删除 |
| 格式 | `key=value`，按第一个 `=` 分割 |

### 7.2 画布参数分类

#### 画布基础

| 参数 | 说明 |
|------|------|
| `canvas_width` | 页面宽度 (像素) |
| `canvas_height` | 页面高度 (像素) |
| `canvas_color` | 背景底色 |
| `canvas_background_image` | 背景图文件名（如 `paper.jpg`），自动缩放到画布尺寸 |

#### 边距

| 参数 | 说明 |
|------|------|
| `margins_top` | 上边距 |
| `margins_bottom` | 下边距 |
| `margins_left` | 左边距 |
| `margins_right` | 右边距 |

#### 网格

| 参数 | 说明 |
|------|------|
| `leaf_col` | 每页列数 |
| `leaf_center_width` | 版心中缝宽度 |
| `leaf_center_linewidth` | 中缝线宽 |

#### 边框

| 参数 | 说明 |
|------|------|
| `outline_width` | 外框线宽 |
| `outline_color` | 外框颜色 |
| `inline_width` | 内框线宽 |
| `inline_color` | 内框颜色 |
| `outline_hmargin` | 边框水平边距 |
| `outline_vmargin` | 边框垂直边距 |

#### 鱼尾装饰

| 参数 | 说明 |
|------|------|
| `if_fishflower` | `0`=三角鱼尾, `1`=弧形花鱼尾 |
| `fish_top_y` | 上鱼尾 Y 坐标 |
| `fish_top_color` | 上鱼尾颜色 |
| `fish_top_rectheight` | 上鱼尾矩形高度 |
| `fish_top_triaheight` | 上鱼尾三角高度 |
| `fish_top_linewidth` | 上鱼尾线宽 |
| `fish_btm_direction` | 下鱼尾方向: `0`=顺(向下), `1`=对(向上) |
| `fish_btm_y` | 下鱼尾 Y 坐标 |
| `fish_btm_color` | 下鱼尾颜色 |
| `fish_btm_rectheight` | 下鱼尾矩形高度 |
| `fish_btm_triaheight` | 下鱼尾三角高度 |
| `fish_btm_linewidth` | 下鱼尾线宽 |
| `fish_flower_image` | 花鱼尾装饰图 (如 `3leaves.png`) |
| `fish_line_color` | 鱼尾装饰线颜色 |
| `fish_line_width` | 鱼尾装饰线宽 |
| `fish_line_margin` | 鱼尾装饰线边距 |

#### 多栏模式

| 参数 | 说明 |
|------|------|
| `if_multirows` | 启用多栏布局 |
| `multirows_num` | 水平栏块数量 |
| `multirows_linewidth` | 分隔线宽 |
| `multirows_colcolor` | 栏分隔线颜色 |
| `multirows_horizontal_layout` | 布局模式: `1`=整页横排(族谱), `2`=半页横排(字典) |

#### Logo

| 参数 | 说明 |
|------|------|
| `logo_text` | 工作室/书坊名称文字 |
| `logo_y` | Logo 纵向位置 |
| `logo_color` | Logo 颜色 |
| `logo_font` | Logo 字体 |
| `logo_font_size` | Logo 字号 |
| `logo_image` | Logo 图片文件 (如 `logo.png`) |

### 7.3 画布渲染步骤

```
1. 创建基础 Image::Magick 对象
   ├─ 有背景图 → 读取 + 缩放
   └─ 无背景图 → 纯色填充

2. 创建透明叠加层 (线条专用)

3. 绘制边框
   ├─ 内细框 (矩形)
   ├─ 外粗框 (矩形偏移)
   └─ 分栏线 (垂直线, 版心中心间距更宽)

4. 多栏模式 → 绘制水平分隔线

5. 绘制鱼尾
   ├─ 上鱼尾 (五边形路径 + 装饰线)
   ├─ 下鱼尾 (方向依 fish_btm_direction)
   └─ 花鱼尾 (缩放+旋转+合成花瓣图)

6. 版心延长线 (从鱼尾延伸至上下边框)

7. 合成叠加层 → 基础画布

8. 绘制底部 Logo
   ├─ 有 logo_image → 缩放 1/3 后合成
   └─ 有 logo_text → 竖排文字

9. 输出 canvas/<id>.jpg
```
